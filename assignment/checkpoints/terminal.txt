# 4 - What is a quick way to create a new folder?

# 5 - What output would you expect when you run the command 'ls' in an empty folder? What about a folder containing files?
In an empty folder, there would be no output. In a full folder, it would show the contents.

# 8 - Why would you run the command 'git status'?
To see if you have any depositories that aren't being tracked by git; and to see the updates made to depositories that are being tracked.

# 9 - What does the command 'git add .' do?
It adds changes to the staging area. When you do the next commit, it adds the changes to main.

# 10 - What is the command to push your committed changes to your repo on GitHub?
